# Snake_Game

🐍 Snake Game

A classic Snake Game built using HTML, CSS, and Vanilla JavaScript.
Currently under development as part of my journey to master core JavaScript concepts.

🎮 Features

Grid-based game board
Smooth snake movement using arrow keys
Random food generation
Wall collision detection (Game Over)
Start & Restart functionality


🧠 What I Learned

Handling game loops using setInterval
Managing state with arrays (snake body)
DOM manipulation for dynamic UI updates
Event handling for keyboard input
Basic game logic implementation


🔧 Future Improvements

Add self-collision detection
Improve UI/UX design
Add difficulty levels (speed increase)

🚧 This project is currently under development and will be improved over time.
